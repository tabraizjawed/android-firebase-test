# Firebase Analytics
Firebase Analytics code guideline for Android developers

## Prerequisites
* Supported Android 4.1 or newer
* Android Studio 3.3.2 or higher
* google-services.json in app-level folder

## Features
* Log events
* Log custom events
* Set user properties
